title: "Directive Shortcuts"
description: "Shorthand properties for common CSS patterns"
---

Popular shorthand keys for common CSS: `p`, `px`, `py`, `m`, `mx`, `my`, `brad`, `bg`, `c`, `w`, `h`, `d`, `fxDir`, `fxJustify`, `fxAItems`, `gap`.

Shorthands are authoring sugar — the generator outputs full CSS behind the scenes. These examples still use plain CSS values (no theming).

## Examples

### Shorthands (simple object)

```html
<div [sj]="{ bg: '#036', c: '#fff', p: '12px', brad: '8px' }">
  bg:#036, c:#fff, p:12px, brad:8px
</div>
```

### Spacing axes (px/py/m)

```html
<div [sj]="{ px: '16px', py: '8px', m: '4px', bg: '#f7f7f7', brad: '6px' }">
  px:16px, py:8px, m:4px
</div>
```

### Responsive shorthands

```html
<div
  [sj]="{
  p: { xs: '8px', md: '16px' },
  brad: { xs: '4px', md: '8px' },
  bg: '#f7f7f7'
}"
>
  p: xs 8px → md 16px, brad: xs 4px → md 8px
</div>
```

### Quick layout (d/flex + fx\*)

```html
<div
  [sj]="{ d: 'flex', fxDir: 'row', fxJustify: 'space-between', fxAItems: 'center', gap: '8px', bg: '#f0f0f0', p: '8px', brad: '6px' }"
>
  <div [sj]="{ bg: '#ccc', p: '4px', brad: '4px' }">A</div>
  <div [sj]="{ bg: '#ccc', p: '4px', brad: '4px' }">B</div>
  <div [sj]="{ bg: '#ccc', p: '4px', brad: '4px' }">C</div>
</div>
```
