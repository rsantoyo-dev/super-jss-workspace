# Padding

Surface components (`sj-paper`, `sj-card`, `sj-flex`) support density presets via `usePadding`.

```html
<sj-paper variant="flat" usePadding="compact">Compact</sj-paper>
<sj-paper variant="flat" usePadding="default">Default</sj-paper>
```

These map to theme surface padding at different breakpoints.
