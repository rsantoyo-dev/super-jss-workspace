# Colors

Reference palette colors via tokens like `primary.main` or `secondary.contrast`.

```html
<div [sj]="sj.bg('primary.main')">Primary</div>
<div [sj]="[ sj.c('secondary.main'), sj.bg('light.main') ]">Secondary text</div>
```

See also: [Palette](./theming-palette)
