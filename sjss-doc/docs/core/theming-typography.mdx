# Typography

Typography variants (`H1`..`H6`, `P`, `SMALL`, `STRONG`, etc.) are defined in the theme and applied via `<sj-typography variant="...">`.

```html
<sj-typography variant="h2">Heading</sj-typography>
<sj-typography variant="p">Body text</sj-typography>
```
